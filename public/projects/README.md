# Imagens Dos Projetos

Substitua os arquivos desta pasta pelos prints reais dos seus projetos.

Arquivos usados hoje:

- navy-commerce.svg
- deepmetrics.svg
- atlas-booking.svg

Voce pode manter os mesmos nomes para trocar direto.

Se preferir outro nome de arquivo, atualize em app/page.jsx no array projects (campo image).

Recomendacao:

- resolucao: 1600x900 ou maior
- formato: .jpg, .png ou .webp
- manter proporcao horizontal para melhor encaixe no card

Links do card:

No app/page.jsx, no array projects, preencha:

- demoUrl
- repoUrl

Quando os campos estiverem vazios, o card mostra o estado "adicionar URL" automaticamente.
